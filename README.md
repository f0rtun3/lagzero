# lagzero

An AI-powered incident intelligence system for event-driven architectures

Your core observability system that:

- Understands streams, not just services
- Correlates:
  - events
  - logs
  - metrics
  - deployments
- Produces:
  - root cause
  - impact analysis
  - recommended actions

At a high level, LagZero has 5 core layers:

- Data Ingestion
- Stream Processing (Real-time intelligence)
- AI Reasoning Layer
- Incident Engine
- Action & Integration Layer
